# Compiling files

Date: 1 Sept 2025

The following commands compile the files to pdf and html formatting. The files are automatically stored in the website directory (`docs/`).
``` bash
 quarto render 1_introToPython_solutions.qmd --to html
 quarto render 1_introToPython_solutions.qmd --to pdf
 quarto render 1_introToPython.qmd --to pdf
 quarto render 1_introToPython.qmd --to html
```
